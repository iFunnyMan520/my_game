### Description

```
This is my first game with PyGame. (-> - move right,
                                    <- - move left,
                                    esc - pause)
```

### Prepare My_Game
```
git clone https://github.com/iFunnyMan520/my_game.git
python3.7 -m venv venv 
source venv/bin/activate
pip install -r requirements.txt
```
